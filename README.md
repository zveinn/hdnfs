# hidden file system
This hides your files.

# How To 
# ./hdnfs --help

# Encryption key
```
export HDNFS=[YOUR 32 BYTE KEY]
```
# Building
 - $ goreleaser release --clean ( add GITHUB_TOKEN )
 - $ goreleaser build --snapshot --clean
